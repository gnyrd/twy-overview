# How do class recordings get processed?

Everything happens automatically after your Zoom class ends.

## The automatic process

1. **Recording downloads.** After the Zoom call ends, the recording downloads automatically.
2. **Thumbnails generated.** A preview image is created from the recording.
3. **Clips extracted.** Short clips are pulled from the full recording automatically.

## Where to find them

Recordings, thumbnails, and clips are stored on the server and uploaded to HeyMarvelous.
You do not need to do anything. It all happens on its own.

## If a recording is missing

Check that the Zoom recording was enabled for that class. If it was and the recording still did not appear, let JP know.
