# Trimming and Publishing

The trim editor lets you set the start and end points of a class recording, then publish it — with intro and outro bumpers — to the on-demand library in one step.

## Opening the Trim Editor

From the calendar at **classes.tiffanywoodyoga.com**:

1. Find the class date
2. Click the **camera icon** (🎥) on the day cell
3. The trim editor loads with the video, waveform, and transcript

## Setting IN and OUT Points

The editor shows three synchronized views:

- **Video player** — scrub or use keyboard shortcuts to find the right moments
- **Waveform** — a visual overview of the audio; red ticks show where students are speaking
- **Transcript** — click any line to jump to that point; each line has IN/OUT buttons

### Using the Waveform

- **Scroll-wheel** to zoom in/out
- **Drag** to pan
- Use the **zoom presets** for different time scales
- **Red ticks** mark student participation (non-host audio)

### Using the Transcript

- Click any line to jump the video to that point
- Click the **IN** or **OUT** button on a line to set it as a trim point

## Publishing

### Trim Only

Click **✂ Trim** to save a trimmed version of the video (with bumpers) without publishing it to the on-demand library. Useful for previewing before going live.

### Trim & Publish

Click **✂ Trim & Publish** to do everything in one step:

1. Creates the class entry in the HeyMarvelous on-demand library
2. Trims the video with intro and outro bumpers
3. Uploads the trimmed video to Vimeo

A progress bar shows the trim and upload status. The whole process typically takes 1–3 minutes depending on class length.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / pause |
| `←` / `→` | Step one frame |
| `⌥←` / `⌥→` | Jump 10 frames |
| `i` | Set IN point |
| `o` | Set OUT point |
| `<` / `>` | Navigate thumbnail candidates |
| `T` | Capture thumbnail at current position |
