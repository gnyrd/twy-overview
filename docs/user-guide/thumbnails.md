# Thumbnails

Thumbnails are the images that represent each class in the HeyMarvelous on-demand library and on Vimeo. They're generated automatically — you just pick the one you want.

## How It Works

After a class recording is processed, the system automatically generates:

- **6 thumbnail candidates** — frames extracted from the video with text overlay (class title, apex pose, date)
- **AI ranking** — candidates are ranked by visual quality, with the best options highlighted

## Choosing a Thumbnail

### From the Trim Editor (Recommended)

1. Open the trim editor for the class (click the 🎥 icon on the calendar)
2. **Gold markers** on the timeline show AI-ranked candidates — brighter = higher ranked
3. Use `<` / `>` keys to navigate between candidates
4. Press `T` at any point to capture the current frame as a thumbnail — text overlay is added automatically
5. A preview appears in the side panel
6. Click **Publish Thumbnail** (amber button) to push it to Vimeo and HeyMarvelous

You're not limited to the AI candidates — press `T` at any moment in the video to use that frame.

### From the Plan Detail Page

1. Open the class plan on the calendar (click the date)
2. If thumbnail candidates exist, a grid of options appears
3. Click one to select it
