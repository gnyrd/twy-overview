# Thumbnails

Thumbnails are the images that represent each class in the HeyMarvelous on-demand library and on Vimeo. They're generated automatically — you just pick the one you want.

## How It Works

After a class recording is processed, the system automatically generates:

- **6 thumbnail candidates** — frames extracted from the video with text overlay (class title, apex pose, date)
- **AI ranking** — candidates are ranked by visual quality, with the best options highlighted

## Choosing a Thumbnail

### From the Trim Editor (Recommended)

1. Open the trim editor for the class (click the video camera icon on the calendar)
2. **Gold markers** on the timeline show AI-ranked candidates — brighter gold = higher ranked
3. Use `<` / `>` keys to navigate between candidates; the timestamp input in the **Thumb** row shows the current candidate's time
4. To jump to a specific time, type a timestamp (e.g. `0:41:16`) into the **Thumb** timestamp field and press Enter
5. Press `T` (or click **"T" — Set**) at any point to capture the current frame as a thumbnail — text overlay is added automatically; the button shows **✓ Selected** once captured
6. After running **Trim & Publish**, the **Publish** button (amber) in the thumbnail row becomes available — click it to push the thumbnail to Vimeo and HeyMarvelous

You're not limited to the AI candidates — seek to any moment in the video and press `T` to use that frame. A thumbnail must be selected before you can run Trim & Publish.

The selected thumbnail is saved to the class plan. If you reload the editor, the thumbnail selection and its timestamp are automatically restored.

### From the Plan Detail Page

1. Open the class plan on the calendar (click the date)
2. If thumbnail candidates exist, a grid of options appears
3. Click one to select it
