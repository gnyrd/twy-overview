# Social Media Clips

After each class, social media clips are created automatically from the recording. AI identifies the best moments — teaching highlights, meditation passages, philosophical insights — and produces vertical video clips with captions, ready for Instagram Reels and similar platforms.

## What Gets Created

For each class, the system produces:

- **10–20 clips** from the best moments (typically 8–50 seconds each)
- **Captions** burned into each clip (speech-to-text, auto-generated)
- **Both captioned and uncaptioned versions** so you can choose
- **A class thumbnail** from the first captioned clip

Clips are in **9:16 vertical format**, ready for Instagram, TikTok, or YouTube Shorts.

## Browsing Clips

Open **[clips.tiffanywoodyoga.com](https://clips.tiffanywoodyoga.com)** to browse all clips.

### Class List

The main page shows recent classes with:

- Class name and date
- Number of clips
- Class thumbnail
- Participant count

Click a class to see its clips.

### Clip Detail

Each clip shows:

- **Classification** — what type of content (philosophy, meditation, teaching, wisdom, etc.)
- **Duration** — clip length in seconds
- **Transcript** — the spoken text
- **Video preview** — play the clip directly in the browser

You can switch between **captioned** and **uncaptioned** versions using the tabs at the top.

### Downloading Clips

Click the **download button** on any clip to save it to your device. Download whichever version you need — captioned for most social media, uncaptioned if you want to add your own text.

## Timing

Clips are typically ready **30–60 minutes** after a class ends (see [Recordings](downloading-recordings) for the full timeline).

Classes older than 6 weeks are automatically cleaned up to save space — the clip metadata and transcripts remain visible on the dashboard, but the video files are removed.
