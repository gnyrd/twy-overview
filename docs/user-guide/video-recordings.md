# How do class recordings get processed?

Everything happens automatically after your Zoom class ends.

## The automatic pipeline

1. **Download** -- Zoom recordings download to the server within 30 minutes of class ending.
2. **Thumbnails** -- The system picks the best frame from the video as a thumbnail. This happens within an hour.
3. **Clips** -- Short highlight clips are extracted with automatic captions.
4. **Archive** -- Recordings are archived to long-term storage overnight.

## Private sessions

Private-client sessions are handled separately:
- Downloaded automatically from Zoom.
- Uploaded to each client's Google Drive folder daily.
- Cleaned up locally after 7 days.

## What you need to do

Nothing. The entire pipeline is automated. If a recording is missing or a thumbnail looks wrong, contact JP.
